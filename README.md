# Firewall Rule Simulator

A Python-based Firewall Rule Simulator that demonstrates how firewall rules allow or block network traffic based on predefined security policies.

## Features

* Simulates firewall rule processing
* Allows or blocks traffic based on rules
* Supports source and destination filtering
* Demonstrates firewall decision making
* Beginner-friendly implementation

## Technologies Used

* Python 3

## Project Structure

firewall_simulator.py

README.md

## Usage

Run the simulator:

python3 firewall_simulator.py

Follow the prompts to test traffic against firewall rules.
